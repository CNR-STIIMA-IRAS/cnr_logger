Detailed Description

File containing the macro definition.

Author
Nicola Pedrocchi
Date
25 Jun 2020 The macro have been designed to follow the basic log4cxx structure (and ROS). The further macro CNR_TRACE_START and CNR_RETURN_xxx are used to trace the input and the output values of the functions

}\
} while (false)

)
Value:
do\
{\
if (ret) { CNR_INFO_COND(logger, std::string(__VA_ARGS__).length() > 0, std::string(__VA_ARGS__)); }\
else { CNR_ERROR_COND(logger, std::string(__VA_ARGS__).length() > 0, std::string(__VA_ARGS__)); }\
CNR_TRACE(logger, (ret ? "[ DONE] " : cnr_logger::RED() + "[FAILED] " + cnr_logger::RESET()) << __FUNCTION__);\
} while (false);\
return ret;\
